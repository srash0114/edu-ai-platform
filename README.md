# edu-ai-platform
1. live demo: https://edu-ai-platform-deploy.vercel.app/

2. chạy ứng dụng này trên máy cục bộ:

Clone Repository:
```
git clone https://github.com/srash0114/edu-ai-platform.git
cd edu-ai-platform
```
Cài đặt và chạy ứng dụng
```
npm install
npm run dev
```
